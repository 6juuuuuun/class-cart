# class_cart 콘텐츠 템플릿

> 마케팅 심리학 기반 — 쓰레드/X 게시용

---

## 캐러셀 이미지 구성 (4장)

| 슬라이드 | 역할 | 디자인 |
|---------|------|--------|
| 1 - Hook | 스크롤 멈추기 | 다크 배경 + 후킹 헤드라인 + 뱃지 |
| 2 - Product | 제품 소개 | 클린 배경 + 제품 사진 + 핵심 장점 |
| 3 - Lifestyle | 공감/신뢰 | 사용 장면 + 후기 인용문 |
| 4 - CTA | 행동 유도 | 댓글 유도 질문 + 브랜딩 |

- 비율: 4:5 세로형 (1080x1350px)
- 일관된 디자인 스타일 유지 (Cormorant Garamond + Inter, 골드 #C9A962)

---

## 톤 & 보이스

| 원칙 | 설명 |
|------|------|
| 판매자가 아닌 발견자 | "써보고 좋아서 알려주는 사람" |
| 반말 + 존댓말 혼합 | "~임", "~거든" 기반, 포인트에서 "~해보세요" |
| 구체적 숫자 | "좋다" 대신 "2주 만에", "리뷰 4,800개", "만원대" |
| 30자 후킹 | 첫 줄 30자 이내, 스크롤 멈추게 하는 한 문장 |
| CTA는 부드럽게 | "궁금하면 댓글", "저장해두세요", "프로필에서 확인" |

## 심리학 기반 6가지 후킹 패턴

### 패턴 1: 반전 고백형
**심리**: 처음 부정 → 반전하면 설득력 급상승 (전환 효과)

```
솔직히 처음엔 {부정적 반응}.
{의심한 이유}.
근데 {사용 계기}
{구체적 결과}.
인정할 건 인정해야지. {핵심 한마디}.
```

**예시 (정관장 에브리타임):**
```
솔직히 홍삼 같은 거 안 믿는 편이었음.
약도 아닌데 뭐가 달라지겠어 싶었는데,
아침마다 커피 3잔 마시다 오후 3시면 또 죽는 패턴이
한 달째 에브리타임 먹고 나서 확 줄었음.
카페인 줄이고 싶으면 이거 한번 써봐.
인정할 건 인정한다.
```

---

### 패턴 2: 일상 침투형
**심리**: "어 이거 나인데?" → 공감 → 방어막 해제

```
{누구나 겪는 일상 상황}.
{그때 느끼는 감정/불편함}.
{제품이 그 순간을 바꿔준 경험}.
{변화된 일상의 구체적 장면}.
```

**예시 (규조토 발매트):**
```
월요일 아침. 겨우 일어나서 샤워했는데
화장실 바닥이 또 축축함. 수건은 눅눅하고 냄새남.
규조토 발매트 깔고 나서 이 스트레스가 사라짐.
밟는 순간 물기 흡수. 빨래 제로.
이런 게 삶의 질이지.
```

---

### 패턴 3: 비교 파괴형
**심리**: "싸게 잘 산 나" → 현명한 소비의 쾌감

```
{비싼 대안}에 {금액/경험}.
근데 {이 제품}이 {가격}인데
{비교 불가 장점}.
{결론: 왜 진작 몰랐지}.
```

**예시 (무선 충전 패드):**
```
카페 가서 맨날 콘센트 자리 전쟁하는 사람.
보조배터리 들고 다니면 가방 무겁고.
이 무선충전패드 만원대인데 책상 위에 딱 두면
올려놓기만 해도 충전됨. 선 없는 책상의 쾌감.
카페비 아끼는 건 덤.
```

---

### 패턴 4: 조용한 경고형
**심리**: 은근한 불안감 → 방어기제 없이 행동 유도 (손실 회피)

```
{평소 신경 안 쓰는 것} 이야기 하나 할게.
{무시하면 생기는 일 / 조용히 진행되는 손실}.
{인식 전환 포인트}.
{제품이 그걸 어떻게 해결하는지}.
```

**예시 (정관장 에브리타임):**
```
아침마다 커피 3잔 마시는 거,
본인은 습관이라고 생각하겠지만
몸은 이미 카페인에 내성 생겨서 효과도 없는 거임.
피로는 그대로인데 위만 상하는 중.
홍삼 에브리타임 한 포가 커피 3잔보다 낫더라.
```

---

### 패턴 5: 선물 추천형
**심리**: "누군가를 위한 소비" → 자기 합리화 가장 쉬움

```
{중요한 사람}한테 뭐 사줘야 하는데 고민인 사람?
{선물 고르기 어려운 상황 공감}.
{이 제품이 선물로 좋은 이유}.
{받은 사람의 반응}.
```

**예시 (안마기):**
```
부모님 생신인데 매년 뭘 드려야 할지 모르겠는 사람.
건강식품은 이미 많고, 옷은 취향 타고.
이 안마기 드렸더니 아버지가 매일 저녁 쓰심.
"이건 돈 아깝지 않다"라는 아버지의 첫마디.
부모님 선물 고민 끝.
```

---

### 패턴 6: 발견 공유형
**심리**: "남들이 모르는 걸 아는 나" → 독점 정보의 우월감 + 호기심

```
{발견 상황 — 우연히/조사해서}.
{제품의 숨겨진 장점이나 비하인드}.
{직접 써본 결과}.
{이건 알려줘야 할 것 같아서}.
```

**예시 (차/음료):**
```
유튜브 알고리즘이 갑자기 이 차(茶) 영상을 띄워줬는데
일본에서 카페비로 한 달에 10만원 쓰는 사람들이
집에서 이걸로 대체한다는 거임.
호기심에 사서 마셔봤는데... 진짜 카페 맛남.
한 박스에 만원대. 이건 알려줘야 될 것 같아서 올림.
```

## 카테고리별 최적 패턴 매칭

| 카테고리 | 1순위 | 2순위 |
|----------|-------|-------|
| 건강기능식품 | 조용한 경고형 | 반전 고백형 |
| 뷰티/스킨케어 | 일상 침투형 | 비교 파괴형 |
| 생활용품/리빙 | 발견 공유형 | 선물 추천형 |
| 식품/음료 | 발견 공유형 | 일상 침투형 |
| 가전/테크 | 비교 파괴형 | 반전 고백형 |
| 패션/잡화 | 일상 침투형 | 발견 공유형 |

## 절대 하지 말 것

- "지금 구매하세요", "최저가" 같은 쇼핑몰 톤
- 과도한 이모지/느낌표
- 호기심만 열고 안 닫기 (알고리즘 페널티)
- 모든 글에 링크 넣기 (5개 중 1개만)
- 같은 패턴 복붙 반복 (매크로 계정 취급)
