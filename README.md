# class_cart 프로젝트 구조

```
partners/
├── strategy.md              # 전략 문서 (지속 업데이트)
├── README.md                # 프로젝트 구조 설명
│
├── contents/                # 게시 콘텐츠 보관
│   ├── threads/             # 쓰레드 게시물 (.pen + 텍스트)
│   └── x/                   # X 게시물
│
├── templates/               # 콘텐츠 템플릿 (.pen)
│
├── assets/                  # 공용 리소스
│   └── images/              # 이미지 소스
│
├── automation/              # API 자동 게시 스크립트
│
└── archive/                 # 게시 완료된 콘텐츠 아카이브
```

## 콘텐츠 파일 네이밍 규칙

```
{날짜}_{카테고리}_{제품키워드}.pen
예: 2026-03-15_생활꿀템_무선충전패드.pen
```

## 카테고리 코드

| 코드 | 카테고리 |
|------|----------|
| 생활꿀템 | 삶의 질 올려주는 제품 |
| 시즌 | 계절/날씨 맞춤 제품 |
| 문제해결 | 불편함 솔루션 |
| 트렌드 | SNS 바이럴/신제품 |
| 가성비 | 가격 대비 성능 발견 |
