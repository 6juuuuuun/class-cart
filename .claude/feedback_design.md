디자인 시 여백을 최소화하고 캔버스를 꽉 채워야 함. "여백 자랑"은 절대 금지.

**Why:** 인스타/쓰레드 피드에서 썸네일로 봤을 때 텍스트가 바로 읽혀야 어그로가 끌림. 여백이 많으면 미완성처럼 보이고 스크롤 패스당함.

**How to apply:**

### 폰트 사이즈 (확정)
- 타이틀: **120px**, fontWeight 900, lineHeight 1.15
- 서브 설명 (테마 컬러 텍스트): **60px**, fontWeight 500~600, lineHeight 1.7
- 키워드 (테마 컬러 강조): **40px**, fontWeight 700
- 태그 필: **30px**, fontWeight 700, padding [14,28], cornerRadius 28
- 가격: **48px**, fontWeight 900
- 상단바 뱃지/로고: **20px**
- 하단바 넘버링/swipe: **16px**
- CTA 버튼 텍스트: **32px**, padding [20,48], cornerRadius 40
- 브랜드 시그니처: **28px**
- 최소 폰트: 16px (상단바/하단바만. 그 외 최소 28px)

### 간격 (확정)
- **세로 가운데 정렬 필수**: justifyContent:"center" 항상 적용
- **1페이지(Hook) gap: 100px** (제목과 다른 요소 사이 넉넉하게)
- **2~4페이지 gap: 24~32px**
- **가로 배치(horizontal row) gap: 40px** (이미지와 텍스트 사이 여유)
- 줄간격: 타이틀 1.15, 서브 설명 1.7, 본문 1.5

### 레이아웃 원칙
- 솔리드 컬러 배경 + 텍스트 중심 (이미지 배경 위 딤드처리 X)
- AI 일러스트(3D 이소메트릭) + 큰 텍스트 + 태그 필로 캔버스 가득 채우기
- 제품 이미지와 AI 일러스트를 나란히 배치 (horizontal row)
- 다크↔다크↔다크↔테마컬러 색상 리듬 (4번째 슬라이드 반전)
- 카테고리별 테마 컬러 (뷰티=골드, 가전=블루, 생활=그린 등)
- 태그 필: 테마 컬러 배경
- 가격: 태그와 다른 디자인 (대형 텍스트 + 라벨 조합 + swipe 유도)
- 제품 이미지 흰 배경이면 cornerRadius 28 + clip:true 라운딩 처리
- 폰트: Noto Sans KR 통일
- 슬라이드 넘버링(01/04), 상단 카테고리 뱃지(20px), 하단 swipe 유도 통일
- **썸네일에서 텍스트가 바로 안 읽히면 폰트가 작은 것** — 더 키울 것
