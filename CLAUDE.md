# class_cart 프로젝트

쿠팡파트너스 어필리에이트 마케팅 자동화 프로젝트.
쓰레드(Threads) + X(Twitter) 기반 제품 큐레이션 채널.

## 브랜드

- **채널명**: class_cart
- **포지셔닝**: 클라스 있는 장바구니 — 안목 있는 큐레이터가 직접 고른 제품만 담는다
- **톤**: 판매자가 아닌 발견자. 반말+존댓말 혼합. 광고처럼 보이면 실패.

## 핵심 원칙

1. 콘텐츠가 먼저, 링크는 자연스럽게 따라온다
2. "사세요"가 아니라 공감/상황 기반 후킹이 먼저
3. 6가지 심리학 패턴 로테이션 (같은 패턴 연속 사용 금지)
4. 링크는 본문이 아닌 댓글에 고정
5. 5개 게시물 중 1개만 제품 링크, 나머지는 순수 콘텐츠

## 워크플로우 스킬

| 스킬 | 용도 |
|------|------|
| `/pick [키워드]` | 쿠팡 API로 제품 검색 및 선정 |
| `/content [제품명]` | 심리학 기반 텍스트 + 캐러셀 4장 제작 |
| `/post [폴더경로]` | Threads/X API로 자동 게시 |

## 주요 파일

- `strategy.md` — 전략 문서 (지속 업데이트)
- `templates/` — 콘텐츠 템플릿 (텍스트 + 디자인)
- `contents/threads/` — 게시 콘텐츠 보관
- `automation/` — API 스크립트
